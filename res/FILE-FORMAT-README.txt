Assignment 10: csv file format
	1. loadPortfolio() and savePortfolio() will load and save a csv file in this kind of format:
	portfolioName, code1, shares1, averageBuyInPrice1, code2, shares2, averageBuyInPrice2..., comissionFeeCounter
	E.g. a.csv is: a,	GOOG,	9.516377686,	1050.82,	FB,	72.50054375,	137.93,	2
	
	2. loadPercentage() and savePercentage will load and save a csv file in this kind of format:
	percentageName, company1, proportion1, company2, proportion2...
	E.g. b.csv is:  b,	goog,	0.2,	twitter,	0.8
